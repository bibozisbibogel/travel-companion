### Data Models for Flight Integration
[Source: architecture/data-models.md]
**FlightOption Entity Requirements**:
- **flight_id**: UUID - Primary identifier for database storage
- **trip_id**: UUID - Foreign key relationship to Trip entity
- **airline**: string - Airline name for user display
- **price**: decimal - Flight price for comparison and budgeting
- **departure_time**: datetime - Departure timestamp for scheduling
- **arrival_time**: datetime - Arrival timestamp for scheduling
- **duration**: integer - Flight duration in minutes for comparison
- **stops**: integer - Number of stops for user preference filtering

### Database Schema Requirements
[Source: architecture/database-schema.md]
**Flight Options Table Schema**:
```sql
CREATE TABLE flight_options (
    flight_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    trip_id UUID REFERENCES trips(trip_id) ON DELETE CASCADE,
    external_id VARCHAR(255), -- Amadeus API provider's ID
    airline VARCHAR(100) NOT NULL,
    flight_number VARCHAR(20),
    origin VARCHAR(10) NOT NULL, -- Airport code
    destination VARCHAR(10) NOT NULL,
    departure_time TIMESTAMP WITH TIME ZONE NOT NULL,
    arrival_time TIMESTAMP WITH TIME ZONE NOT NULL,
    duration_minutes INTEGER NOT NULL,
    stops INTEGER DEFAULT 0,
    price DECIMAL(10,2) NOT NULL,
    currency CHAR(3) DEFAULT 'USD',
    booking_url TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

### Coding Standards Requirements
[Source: architecture/coding-standards.md]
**Python Naming Conventions**:
- **Classes**: PascalCase (`FlightAgent`, `AmadeusClient`)
- **Functions**: snake_case (`search_flights`, `calculate_duration`)
- **API Endpoints**: kebab-case (`/api/v1/flights/search-options`)
- **Database Tables**: snake_case (`flight_options`, `external_id`)

**Critical Rules**:
- **Never log sensitive data**: No API keys, authentication tokens in logs - use structured logging with redaction
- **All external API calls must use circuit breakers**: Prevent cascade failures from Amadeus API outages
- **Database queries must use repository pattern**: Never direct ORM calls from agent classes
- **All async operations require timeout**: Prevent hanging requests from external services (30s default)
