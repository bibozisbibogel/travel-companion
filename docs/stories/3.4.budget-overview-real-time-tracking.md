# Story 3.4: Budget Overview & Real-time Tracking

## Status
Ready for Review

## Story
**As a** budget-conscious traveler,
**I want** clear visibility into my trip costs with real-time budget tracking,
**so that** I can ensure my trip stays within my financial limits.

## Acceptance Criteria
1. Total trip budget vs. estimated cost comparison with visual progress bar
2. Category breakdown: flights, accommodation, activities, dining, transportation, misc.
3. Daily budget allocation showing cost distribution across trip days
4. Currency display matching user's selected currency type
5. Visual indicators (green/yellow/red) for budget status (under/at/over budget)
6. Cost optimization suggestions when over budget
7. Ability to adjust budget and see updated recommendations
8. Export budget summary as PDF or CSV
9. Modern minimal design with clear data visualization
10. Mobile-responsive budget cards and charts

### Relevant Architecture Information
[Source: docs/architecture/tech-stack.md]
- **Frontend Framework**: Next.js 14 with React 18
- **UI Component Library**: shadcn/ui for budget cards
- **Charting Library**: Recharts or Chart.js for budget visualizations
- **PDF Export**: jsPDF or react-pdf for PDF generation
- **CSV Export**: Papa Parse or custom CSV generation
- **State Management**: React Context or Zustand for budget state

### API Integration
[Source: docs/architecture/api-design.md]
- **Get Budget Endpoint**: `GET /api/trips/:tripId/budget` - Fetch trip budget details
- **Update Budget Endpoint**: `PATCH /api/trips/:tripId/budget` - Update budget amount
- **Get Optimization Suggestions**: `GET /api/trips/:tripId/budget/optimize` - Get cost reduction suggestions

### Data Models
[Source: docs/architecture/data-models.md and Story 2.6]
- **Budget Model**:
  - total_budget: decimal
  - currency: string
  - estimated_cost: decimal
  - remaining_budget: decimal
  - status: enum (under, at, over)
- **CategoryCost Model**:
  - category: enum (flights, accommodation, activities, dining, transportation, misc)
  - amount: decimal
  - percentage: decimal
- **DailyCost Model**:
  - date: date
  - total_cost: decimal
  - category_breakdown: CategoryCost[]
- **OptimizationSuggestion Model**:
  - suggestion_type: string (lower_cost_activity, cheaper_accommodation, remove_activity)
  - description: string
  - potential_savings: decimal
  - action_data: object

### Design System
- **Color Palette**:
  - Budget Under: Green (#10B981)
  - Budget At: Yellow (#F59E0B)
  - Budget Over: Red (#EF4444)
  - Category Colors: Use distinct accessible colors for each category
- **Typography**: Clear hierarchy, bold for totals, regular for details
- **Charts**: Simple, clean charts with minimal gridlines and axes
- **Cards**: White background, subtle shadows, rounded corners

### Budget Calculation Logic
[Source: Story 2.6 Itinerary Agent]
- Sum all flight costs from selected flights
- Sum accommodation costs (price_per_night × number_of_nights)
- Sum activity costs from all selected activities
- Sum restaurant costs from meal recommendations
- Add estimated transportation costs between locations
- Calculate miscellaneous as 10% of total or fixed amount

### Export Formats
- **PDF Format**:
  - Header: Trip name, dates, destination
  - Budget Overview: Total budget, estimated cost, status
  - Category Breakdown: Table with categories, amounts, percentages
  - Daily Breakdown: Chart or table showing daily costs
  - Footer: Export date, page numbers
- **CSV Format**:
  - Columns: Date, Category, Item Name, Amount, Currency
  - Include summary rows for totals
  - Easy import into spreadsheet software

- **Testing Framework**: Jest + React Testing Library
- **Test Coverage**: Minimum 80% for budget components

- **Export Tests**: Validate PDF and CSV output structure
